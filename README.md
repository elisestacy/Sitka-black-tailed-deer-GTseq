# Sitka-black-tailed-deer-GTseq
Assessing the application of a Sitka black-tailed deer GTseq SNP panel across Alaska and Haida Gwaii, B.C.
## Bioinformatic processing scripts 
Processing and initial genotyping of GT sequencing data follows Kira Long's reference aligned pipeline https://github.com/kiralong/gtseq_ref_align
## R Markdown script
Details further genotype filtering, data analysis methods, and results visualization
