# Sitka-black-tailed-deer-GTseq
Assessing the application of a Sitka black-tailed deer GTseq SNP panel across Alaska and Haida Gwaii, B.C.
Bioinformatic processing scripts for GT sequencing data follows Kira Long's reference aligned pipeline https://github.com/kiralong/gtseq_ref_align
R Markdown script details data analysis and results visualization
